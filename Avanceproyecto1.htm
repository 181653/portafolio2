Definición y planificación del sitio web.

 

Objetivos:

Desarrollar una pagina web que contenga los elementos aprendidos de html

Definición de objetivos:

·       Brindar información respecto al videojuego en cuestión.

·       Ser un medio de entretenimiento.

·       Brindar contenido multimedia que complemente el blog.

 

Caracterización del público:

El público al que esta enfocado es principalmente una audiencia de 17 años a 40, sin embargo, es apto para todo público.

 

Selección y conformación del equipo de trabajo:

El trabajo se realizará por una única persona (Erik del Angel Arriaga Torres), utilizando equipo de computo tanto el que se brinda en la universidad como el equipo personal, la pagina se desarrollara mediante programas como visual estudio code (esto para desarrollar la interfaz y los elementos que la pagina tendrá así como darle estilo a esta misma.

 

Arquitectura de la información

 

El tipo de estructura:

Es una página con el tipo de estructura red ya que las diferentes secciones de esta están entre lazados.

Especificaciones de diseño:

La página se manejara con una estructura de red a su vez esta tendrá distintos tipos de estilos insertados mediante visual estudio y conteniendo multimedia e información referente al tema.
